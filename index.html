Task 1

Create a table named products with the following fields: id: Unique integer, auto-increment, primary key. product_name: String (VARCHAR) with a maximum length of 100, cannot be null. price: Decimal

Code:

CREATE TABLE products (id INT(3) UNIQUE AUTO_INCREMENT PRIMARY KEY, product_name VARCHAR(100) NOT NULL, price FLOAT(7,4) NOT NULL);

HERE IS THE QUERY STATEMENT AND TABLE STRUCTURE

Task 2

Add a CHECK constraint to ensure that the price of the product must be greater than 0

Code:

ALTER TABLE products MODIFY COLUMN price FLOAT(7,2) CHECK (price > 0);
